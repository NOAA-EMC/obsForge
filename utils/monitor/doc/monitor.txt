Obsforge monitoring system on hpc
---------------------------------
the monitoring system uses sqlite3 database to 
maintain and update inventory,
inspect and qc the data, 
compute derived data products
and generate a website.
 
the input is the root of the data directory tree.
the system can be run in a pipeline on the tree
and it can also be run as a cron job to track data generation.
Data is produced in 4 cycles per day (00,06,12,18)
there are several datasets (gdas, gfs, gcdas, ...)
each  dataset is produced by several tasks;
each task produces data for several observation categories;
each obs category consist of several obs spaces;
